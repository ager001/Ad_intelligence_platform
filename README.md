# Ad Intelligence Platform

## Goal

Predict whether a user clicks an advertisement.

## Concepts Covered

- Bernoulli Distribution
- Binomial Distribution
- Gaussian Distribution
- Poisson Distribution
- Logistic Regression
- Gaussian Naive Bayes
- Maximum Likelihood Estimation

## Dataset

Ad Click Prediction Dataset from Kaggle.

## Business Objective

Increase advertising conversion rates through predictive analytics.
